# C++模板函数与重载函数

## 模板函数

可以理解为一段通用代码  
形参类型由编译器决定  
下述类型要相同，不同编译器会报错  

```cpp
template<class T>

T function(T a, T b, T c)
{
    return a + b * c;
}

```

更通用的版本  

```cpp
template<class Ta, class Tb, class Tc>

Ta function(Ta a, Tb b, Tc c)
{
    return a + b * c;
}

```

## 重载函数

定义多个同名函数的机制称为函数重载  

```cpp
void function(int a, int b)
{
    a = b;
}

void function(int a, int b, int c)
{
    a = b + c;
    a = b;
}

void function(float a, float b, float c)
{
    a = b + c;
    a = b;
}

```
  
------
模板函数和重载函数有什么不同  

* 模板函数只能定义一个
* 重载函数可以多个
* 模板函数参数数量固定，而重载函数可以不定
